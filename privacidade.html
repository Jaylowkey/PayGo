Informações que Coletamos

Coletamos apenas as informações necessárias para fornecer nossos serviços de forma segura e eficiente.

Dados Pessoais

  • Nome completo
  • Número de telefone (M-Pesa/e-Mola)
  • Endereço de email
  • Histórico de transações

Dados Técnicos

  • Endereço IP
  • Tipo de dispositivo e navegador
  • Dados de cookies e analytics
  • Logs de acesso
2

Como Usamos suas Informações

Suas informações são utilizadas exclusivamente para:

Processar Pagamentos

Para completar suas transações internacionais de forma segura

Comunicação

Para enviar confirmações, actualizações e suporte ao cliente

Segurança

Para prevenir fraudes e proteger sua conta contra acessos não autorizados

Melhorias

Para optimizar nossos serviços e experiência do utilizador

3

Compartilhamento de Dados

Nunca Vendemos seus Dados

Seus dados pessoais nunca são vendidos, alugados ou compartilhados com terceiros para fins de marketing

Podemos compartilhar dados apenas com:

Processadores de pagamento (M-Pesa, e-Mola)

Apenas dados necessários para processar transações

Autoridades legais

Apenas quando exigido por lei ou ordem judicial

Prestadores de serviço

Com acordos de confidencialidade rigorosos

4

Segurança de Dados

Implementamos medidas de segurança robustas para proteger suas informações:

Criptografia SSL

Todos os dados são transmitidos de forma segura com criptografia de ponta

Armazenamento Seguro

Dados guardados em servidores protegidos com múltiplas camadas de segurança

Acesso Restrito

Apenas pessoal autorizado tem acesso aos dados dos clientes

Actualizações

Sistemas de segurança sempre actualizados contra novas ameaças

5

Seus Direitos

Você tem os seguintes direitos sobre seus dados pessoais:

Acesso

Solicitar uma cópia dos dados que temos sobre você

Correcção

Solicitar correcção de dados incorrectos ou incompletos

Eliminação

Solicitar eliminação dos seus dados (com ressalvas legais)

Oposição

Opor-se ao processamento dos seus dados em certas circunstâncias

Para exercer estes direitos, contacte-nos via suporte@paygo.co.mz ou WhatsApp +258 83 752 2255.
